Projeto Next.js

Este é um projeto desenvolvido com Next.js utilizando o comando create-next-app.

Como iniciar o projeto

Primeiro, execute o servidor de desenvolvimento:

npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev

Depois, abra http://localhost:3000 no navegador para visualizar o projeto.

Você pode começar a editar a página modificando o arquivo:

app/page.tsx

A aplicação será atualizada automaticamente conforme as alterações forem salvas.
